= Overview =
This is a companion plugin for the PandoWP theme, it add all functionalities that are plugin-territory to the PandoWP theme.

PandoWP is currently under review on wordpress.org: https://themes.trac.wordpress.org/ticket/50201.


= Installation Instructions =
1. Upload `pando-extra` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
